Covid-19 Vaccines

For a few weeks I have tried to keep up with the draft vaccine landscape documents published by the WHO. I figured that if I do this work anyway I might as well bundle the dataset in an R package. The package is called covid19vaccines and contains various updates of the WHO Draft Landscape in a tidy table.

The package homepage has installation details and an example. I also added a Shiny web app that makes it somewhat easier to filter through the table.

I hope to be able to update the package each week. In building the package I profited from a number of sources:

This post by David F. Kleinschmidt.

Kieran Healy’s beautifully simple nycdogs package.
